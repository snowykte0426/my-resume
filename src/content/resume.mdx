  - 3계층 분리로 책임 명확화, Port-Adapter 패턴으로 외부 의존성 유연화
- 메타데이터 기반 DB 스키마 설계로 평가 항목 변경 유연 대응
- Caffeine 캐싱으로 API 최적화 (**평균 669ms → 352ms, 47.4% 향상**)

- Kotest 기반 테스트 코드, **커버리지 75%+**
#### 팀 리더십
- 데일리 스크럼 및 스프린트로 협업 환경 조성
- Exposed ORM 등 기술 스터디 세션 진행
#### 개발 환경 및 자동화
- <b>[AWS CodeDeploy Hooks 모방 GitHub Action](https://github.com/marketplace/actions/ssh-remote-deploy)</b> 개발로 SSH 환경 배포 간소화
- GitHub Actions CI/CD 파이프라인 구축

2025학년도의 성공적인 운영으로 익년에도 사용되기로 결정되어 프로젝트를 이어나가게 되었습니다, <br />
